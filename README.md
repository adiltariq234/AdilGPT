# 🚀 AdilGPT

A production-ready **Agentic AI Assistant** built with **LangGraph, LangChain, FastAPI, RAG, SQLAlchemy, and LangSmith**.

---

# 📦 Getting Started

Follow the steps below to run **AdilGPT** on your local machine.

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/adiltariq234/AdilGPT.git
```

---

## 2️⃣ Navigate to the Project Directory

```bash
cd AdilGPT
```

---

## 3️⃣ Create a Virtual Environment (Optional but Recommended)

Using Conda:

```bash
conda create -n adilgpt python=3.11 -y
```

---

## 4️⃣ Activate the Virtual Environment

```bash
conda activate adilgpt
```

---

## 5️⃣ Install the Required Dependencies

```bash
pip install -r requirements.txt
```

> **Note:** Make sure the file is named `requirements.txt`.

---

## 6️⃣ Configure Environment Variables

Create a `.env` file in the project root and add your API keys:

```env
GROQ_API_KEY=your_api_key
LANGSMITH_API_KEY=your_api_key
OPENAI_API_KEY=your_api_key
```

---

## 7️⃣ Run the Application

```bash
python app.py
```

---

## 🎉 You're Ready!

Open the application in your browser and start chatting with **AdilGPT**.
